API Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Create a new user /users POST { "name": "Charlie" } { "id": "3", "name": "Charlie", "favouriteMovies": [] }
Get a list of all movies /movies GET None A JSON object holding data about all movies.
Get data about a single movie by title /movies/Example%20Movie%20Title

e.g. /movies/Amélie 		GET None A JSON object holding all data about the target movie "Amélie".
Get data about a genre by name of genre /movies/genre/Example%20Genre

e.g. /movies/genre/Romantic%20Comedy 		GET None {
"name": "Romantic Comedy",
"description": "Romantic comedies are a subgenre of comedy movies that focus on the romantic relationships between characters. They often involve humorous situations and light-hearted storytelling."
}
Get data about a director by name /movies/directors/Name%20Of%20Director

e.g. /movies/directors/Jean-Pierre%20Jeunet 		GET None {
"name": "Jean-Pierre Jeunet",
"bio": "Jean-Pierre Jeunet is a French filmdirector known for his distinctive visual style. He gained international acclaim for directing the film 'Amélie.'",
"dob": "03/09/1953"
}
Update user information (username) /users/{UserId}

e.g. /users/1 		PUT { "name": "Denise" } { "id": "1", "name": "Denise", "favouriteMovies": [] }
Add a movie to a user's favorite movies /users/{UserId}/Example%20Movie%20Title}

e.g. /users/2/Amélie 		PUT None A text messeage indicating whether the movie was successfully added.
Remove a movie from a user's favorite movies /users/{UserId}/Example%20Movie%20Title}

e.g. /users/2/Amélie 		DELETE None A text messeage indicating whether the movie was successfully removed.
Deregister a user /users/{UserId}}

e.g. /users/2 		DELETE None A text messeage indicating whether the user was successfully removed
